# Code of Conduct

## Our pledge

We want LAI to be a welcoming, respectful, and constructive open source project.

Participants are expected to:

- be respectful and patient
- assume good intent
- give actionable feedback
- keep technical disagreements focused on the work

## Unacceptable behavior

- harassment or personal attacks
- discrimination or exclusionary language
- publishing private information without permission
- repeated bad-faith or disruptive behavior

## Enforcement

Project maintainers may remove comments, issues, pull requests, or contributors that violate this code of conduct.

## Reporting

If you experience or witness unacceptable behavior, please contact the maintainers privately through GitHub.
